Private Secured Chat System

In development work by Truong Minh Hieu

Begin Chatting

Features

Secure 1-on-1 messaging with real-time updates, PBKDF2-based AES encryption, chat history, timestamps, and a clean logout option.

Real-time private messaging with a user login system, live online user list, secure AES encryption using PBKDF2 and CBC mode, encrypted chat history per conversation, message timestamps, and a logout button for session reset.

Tính năng

Nhắn tin 1-1 an toàn với cập nhật theo thời gian thực, mã hóa AES dựa trên PBKDF2, lưu lịch sử trò chuyện, dấu thời gian và tùy chọn đăng xuất gọn gàng.

Nhắn tin riêng theo thời gian thực với hệ thống đăng nhập người dùng, danh sách người dùng trực tuyến trực tiếp, mã hóa AES an toàn sử dụng PBKDF2 và chế độ CBC, lưu trữ lịch sử trò chuyện được mã hóa cho từng cuộc hội thoại, dấu thời gian cho tin nhắn và nút đăng xuất để đặt lại phiên.
